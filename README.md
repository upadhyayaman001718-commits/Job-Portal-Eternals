# Job Portal Eternals
Job Searching Portal

A simple and efficient Job Searching Portal designed to connect job seekers and employers. The portal allows users to create profiles, browse job listings, and apply for suitable jobs, while employers can post vacancies and manage applications. Built with clean architecture and easy navigation, this project demonstrates core concepts like CRUD operations, user management, and data handling.

Features

User registration & login

Job posting and job browsing

Search and filter options

Apply for jobs directly

Employer dashboard

Simple, clean UI
